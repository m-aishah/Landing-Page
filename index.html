Monty Interpreter

The Monty ByteCode Interpreter is a project aimed at creating an interpreter for Monty ByteCodes files.

Monty 0.98 is a scripting language that is first compiled into Monty byte codes, similar to Python. It relies on a unique stack with specific instructions to manipulate it. This project focuses on implementing these instructions in C, allowing users to execute Monty scripts.

Completed: September 2023
